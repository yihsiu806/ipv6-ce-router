NAME

  Router Ignores Invalid Router Solicitations 
  Part A: Hop Limit is not 255

Purpose

  Verify that a router ignores invalid Router Solicitations.

References

[RFC 7084],L-1 [RFC 4861],Section 6.1.1, 6.2.6

Resource Requirements

  * Packet generator
  * Monitor to capture packets

Test Setup

  Common Test Setup is performed at the beginning of this test part.

Procedure

  Part A: Hop Limit is not 255
   1. TN2 transmits a Router Solicitation with an IPv6 Hop Limit of 254. The Router Solicitation is
      valid otherwise.
   2. Observe the packets transmitted by the CE-Router.

Observable Results

  Parts A-F
   In all Parts, the CE-Router must discard the Router Solicitation from TN2 and must not transmit a
   corresponding Router Advertisement within MAX_RA_DELAY_TIME (0.5) seconds.

Possible Problems

  * None.

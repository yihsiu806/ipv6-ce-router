NAME

  Reception of A Reply Message for Prefix Delegation
  Part F: IA Prefix option not included in IA PD from Server

TARGET

  CE-Router

SYNOPSIS

INITIALIZATION

If the Reply was received in response to a Solicit (with a Rapid Commit option), Request, Renew or Rebind message, the client updates lifetimes for any prefixes in the IA option that the client already has recorded in the IA.
Leave unchanged any information about prefixes the client has recorded in the IA but that were not included in the IA from the server.

NAME

  Next-hop Determination

Purpose

  Verify that a host properly determines the next hop.

References

[RFC 7084],W-3 [RFC 4861],Sections 3.1, 5.2 and 6.3.6

Resource Requirements

  * Packet generator
  * Monitor to capture packets

Test Setup

  Common Test Setup 1.2 is performed at the beginning of this test part.
  The Common Test Cleanup procedure is performed after each part.
Router Advertisement A
IPv6 Header
Next Header: 58
Source Address: TR1's Link-local Address
Router Advertisement
Router Lifetime: 600 seconds
Reachable Time: 0 seconds
Retransmit Interval: 1 second
Source Link-layer Option

Echo Request B
IPv6 Header
Next Header: 58
Source Address: TN1's off-link Global Address
Destination Address: CE-Router's Global Address
ICMPv6 Echo Request

Procedure

  1. TR1 transmits Router Advertisement A to the CE-Router.
  2. TN1 transmits Echo Request B to the CE-Router with a nexthop of TR1.
  3. Observe the packets transmitted by the CE-Router.

Observable Results

  Step 4:
  The CE-Router should transmit an Echo Reply to TN1's off-link global address using TR1 as it's
  next hop.

Possible Problems

  * None.
